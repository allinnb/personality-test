## 每种动物的详细档案

### 🦁 狮子 LEADER
**代表人格**：ENTJ / ESTJ
**代表SBTI**：商业型 B > 技术型 T

**自画像**：
> "我就是主角！不管是班级活动还是小组项目，我总是那个站出来承担的人。制定目标、分配任务、推动执行——这是我的天赋。"

**超能力**：领导力 ⚡ 决策力 🛡️ 执行力 💪
**成长盲区**：有时候太强势，听不进去别人的意见
**学习style**：听觉+动觉型居多，喜欢边做边学

**🦁 狮子专属课程推荐**：
- **A-Level**：经济 + 商业研究 + 数学
- **AP**：微观经济 + 心理学
- **IB DP**：商业管理HL + 经济SL
- **BC**：商业管理方向

**🦁 选课建议**：
主攻商科类，数学是基础。可搭配心理学理解人性，不建议纯艺术除非你是ESTP。

**🦁 报考竞争力**：领导力演讲比赛、商业案例分析大赛
**适合社团**：学生会、创业社、商业竞赛队

---

### 🦊 狐狸 CLEVER
**代表人格**：ENTP / INTP
**代表SBTI**：研究型 R > 技术型 T

**自画像**：
> "我对世界充满好奇！宇宙的尽头是什么？为什么这个东西是这样运作的？和别人辩论一个观点可以让我兴奋一整天。我喜欢找到那个'啊哈'的顿悟时刻。"

**超能力**：洞察力 🔍 创新力 💡 逻辑力 🧠
**成长盲区**：想太多却迟迟不行动，容易眼高手低
**学习style**：视觉型居多，喜欢看图表和概念图

**🦊 狐狸专属课程推荐**：
- **IB DP**：数学HL + 物理HL + TOK
- **AP**：计算机科学 + 物理C + 统计学
- **A-Level**：数学 + 高数 + 物理 + 计算机
- **BC**：STEM方向

**🦊 选课建议**：
数学和科学是核心，语言表达能力也要发展（TOK/哲学课）。建议参加科学奥赛或辩论赛锻炼表达。

**🦊 报考竞争力**：数学竞赛、物理碗、辩论赛
**适合社团**：科学社、哲学社、辩论队、机器人社

---

### 🦉 猫头鹰 WISDOM
**代表人格**：INTJ / ISTJ
**代表SBTI**：研究型 R / 技术型 T

**自画像**：
> "我不喜欢说废话。我追求的是把事情做到极致。我的笔记可能是全班最整洁的，我的计划可能是最周密的。在我这里，没有'差不多'这个词。"

**超能力**：分析力 📊 执行力 ✅ 专注力 🎯
**成长盲区**：对不完美容忍度为零，容易给自己和身边人压力
**学习style**：视觉型，笔记和思维导图达人

**🦉 猫头鹰专属课程推荐**：
- **IB DP**：数学HL + 计算机HL + 物理SL
- **AP**：微积分BC + 计算机科学 + 物理
- **A-Level**：数学 + 高数 + 计算机 + 物理
- **BC**：计算机科学方向

**🦉 选课建议**：
STEM路线走到黑。但建议适当选一门人文课（经济学或哲学），训练跨界思维。

**🦉 报考竞争力**：信息学奥赛、数学奥赛、机器学习项目
**适合社团**：编程社、STEM竞赛队、学术期刊编辑

---

### 🦋 蝴蝶 CREATIVE
**代表人格**：INFP / ISFP
**代表SBTI**：艺术型 S

**自画像**：
> "我感受得到别人感受不到的东西。一首歌可以让我流泪，一幅画可以让我发呆。我喜欢用文字、画笔、相机记录这个世界，因为那是我存在的方式。"

**超能力**：感知力 🌈 创造力 🎨 共情力 💛
**成长盲区**：太活在自己世界里，有时候行动力跟不上想法
**学习style**：动觉型+视觉型，创意项目中学得最好

**🦋 蝴蝶专属课程推荐**：
- **IB DP**：视觉艺术HL + 文学HL + 音乐HL（任选其一深耕）
- **AP**：艺术史 + 心理学 + 英语文学
- **A-Level**：艺术设计 + 摄影 + 媒体研究
- **BC**：艺术与设计方向

**🦋 选课建议**：
艺术/文学是你的主战场，但别忘了打好学术基础。建议视觉艺术+心理学/文学的组合。

**🦋 报考竞争力**：艺术作品集、创意写作比赛、摄影展
**适合社团**：文学社、摄影社、戏剧社、艺术社团

### 🐺 狼 PACK
**代表人格**：INFJ / ENFJ
**代表SBTI**：社交型 A

**自画像**：
> "我总能看穿别人在想什么。我被信任是因为我真诚地关心每一个人。站在讲台上分享我的想法时，我觉得自己在发光。我希望这个世界因为我变得更好一点点。"

**超能力**：洞察力 🔮 感染力 🎤 共情力 💜
**成长盲区**：太在意别人的看法，有时候不懂拒绝
**学习style**：听觉型，倾听和讨论中学得最好

**🐺 狼专属课程推荐**：
- **IB DP**：心理学HL + 文学HL + 社会学SL（利他型组合）
- **AP**：心理学 + 英语文学 + 乐理
- **A-Level**：心理学 + 社会学 + 文学
- **BC**：心理学/社会科学方向

**🐺 选课建议**：
文科+社会科学是核心。建议心理学打底，配合理工科增加逻辑性（如AP统计）。

**🐺 报考竞争力**：模拟联合国、志愿服务项目、演讲比赛
**适合社团**：模拟联合国、志愿服务队、心理学社、文学社

### 🐬 海豚 PLAYFUL
**代表人格**：ENFP / ESFP
**代表SBTI**：艺术型 S + 社交型 A

**自画像**：
> " Life is a party! 我是朋友圈的气氛担当，永远有说不完的话和冒不完的点子。我喜欢和人待在一起，喜欢新鲜事物，每一天都要精彩！"

**超能力**：社交力 🎉 创意力 ✨ 适应力 🌊
**成长盲区**：三分钟热度，专注力是永远的痛
**学习style**：动觉型，需要互动和体验式学习

**🐬 海豚专属课程推荐**：
- **IB DP**：戏剧HL + 经济学SL + 心理学SL（活力组合）
- **AP**：美国历史 + 艺术史 + 心理学
- **A-Level**：商业 + 艺术设计 + 心理学
- **BC**：传媒/旅游方向

**🐬 选课建议**：
选择有趣的、可以表达的课程。避免纯理科刷题型，选实践性强的课程更适合你。

**🐬 报考竞争力**：演讲/表演类比赛、社交媒体运营项目
**适合社团**：戏剧社、主持人队、舞蹈社、学生媒体

### 🐼 熊猫 GENTLE
**代表人格**：ISFJ / ESFJ
**代表SBTI**：社交型 A + 艺术型 S

**自画像**：
> "我可能是那个默默记住所有人生日的人。我喜欢照顾别人，被需要让我感到快乐。我不喜欢冲突，我相信和谐的力量。"

**超能力**：照顾力 🫶 耐心力 🧘 观察力 👁️
**成长盲区**：太迁就别人，容易忽略自己的需求
**学习style**：听觉型+视觉型结合，笔记认真

**🐼 熊猫专属课程推荐**：
- **IB DP**：生物HL + 语言SL + 心理学SL（关怀型组合）
- **AP**：生物 + 心理学 + 英语语言
- **A-Level**：生物 + 化学 + 心理学
- **BC**：护理/健康科学方向

**🐼 选课建议**：
生命科学+社会科学方向。生物/护理/教育都是适合你的领域。

**🐼 报考竞争力**：志愿者证明、医院体验项目、教育类实习
**适合社团**：红十会、动物保护社、义工社、教育辅导队

### 🦅 鹰 EAGLE
**代表人格**：ESTP / ISTP
**代表SBTI**：技术型 T + 研究型 R

**自画像**：
> "别跟我废话，我要的是结果。我喜欢动手解决问题，电脑坏了？给我10分钟。数学题太难？直接上答案推演。我享受在实战中学习。"

**超能力**：行动力 🏃 危机处理 🚨 动手力 🔧
**成长盲区**：冲动派，容易忽略长期后果
**学习style**：动觉型，做中学是最佳方式

**🦅 鹰专属课程推荐**：
- **BC**：计算机科学 + 数学 + 物理（硬核STEM）
- **AP**：物理C + 计算机科学 + 微积分
- **A-Level**：数学 + 物理 + 计算机 + 设计技术
- **IB DP**：设计技术HL + 物理SL

**🦅 选课建议**：
STEM动手型。工程、计算机、物理是核心。不要强迫自己学纯文科。

**🦅 报考竞争力**：机器人比赛、编程项目、物理实验竞赛
**适合社团**：机器人社、编程社、航模社、赛车队

### 🐢 乌龟 STEADY
**代表人格**：ISTP / ISFP
**代表SBTI**：技术型 T / 艺术型 S

**自画像**：
> "我不喜欢张扬，但我有自己的节奏。我的世界很丰富，只是我不急着用语言表达。给我一个问题，我会安静地、一步一步地解决它。"

**超能力**：专注力 🎯 动手力 🔧 稳定力 ⛰️
**成长盲区**：慢热型，可能被误解为不积极
**学习style**：动觉型+视觉型，独立研究型学习者

**🐢 乌龟专属课程推荐**：
- **BC**：应用数学 + 计算机 + 物理（扎实路线）
- **A-Level**：数学 + 物理 + 设计技术 + 艺术
- **AP**：艺术设计 + 计算机原理 + 物理
- **IB DP**：设计技术HL + 物理SL

**🐢 选课建议**：
技术+艺术的交叉领域。建筑、工业设计、游戏设计、影视制作都适合你。

**🐢 报考竞争力**：工程设计比赛、艺术作品集、技术项目
**适合社团**：创客空间、摄影社、动画社、建筑社

### 🐯 老虎 COMPETITOR
**代表人格**：ESTJ / ENTJ
**代表SBTI**：商业型 B + 技术型 T

**自画像**：
> "我要赢。我参加比赛就是为了拿第一，我不掩饰我的野心。我相信实力说话，结果证明一切。压力？那是我的燃料。"

**超能力**：竞争力 🏆 执行力 ⚡ 抗压力 🛡️
**成长盲区**：输不起，竞争心太强可能影响人际关系
**学习style**：听觉型+视觉型结合，效率至上

**🐯 老虎专属课程推荐**：
- **A-Level**：经济 + 商业研究 + 数学 + 高数（学霸路线）
- **AP**：微观经济 + 宏观经济 + 统计学 + 微积分
- **BC**：商业管理方向
- **IB DP**：商业管理HL + 经济HL + 数学SL

**🐯 选课建议**：
商科+数学是标配。可以挑战IB HL或AP多门，但注意别只顾竞争忘了合作能力。

**🐯 报考竞争力**：商业竞赛（BPA/FBLA）、数学竞赛、经济学论文
**适合社团**：商业社、辩论队、学生会、竞赛培训班

### 🐨 考拉 CALM
**代表人格**：ISFJ
**代表SBTI**：社交型 A

**自画像**：
> "我不喜欢变化，我喜欢我熟悉的人和事。我可能不是最耀眼的，但我是你最可以信赖的朋友。稳定、温暖、有耐心——这是我的关键词。"

**超能力**：稳定力 🏔️ 耐心力 🧘 忠诚力 💎
**成长盲区**：太安于现状，可能错过突破自我的机会
**学习style**：听觉型，需要稳定的学习节奏

**🐨 考拉专属课程推荐**：
- **IB DP**：生物HL + 语言HL + 环境科学SL（温和组合）
- **A-Level**：生物 + 化学 + 心理学
- **BC**：健康科学/护理方向
- **AP**：生物 + 心理学 + 环境科学

**🐨 选课建议**：
生命科学/健康科学方向。稳定的课程体系更适合你，避免频繁换环境。

**🐨 报考竞争力**：医学预科项目、护理实习、生态调研
**适合社团**：红十会、环保社、动物保护、图书馆志愿者

### 🦄 独角兽 UNIQUE
**代表人格**：特殊组合（如INFJ+ENTP混合、INTJ+ESFP混合等）
**代表SBTI**：混合型（无单一主导，两个以上类型得分接近）

**自画像**：
> "我可能不太符合上面的任何一种描述。我时而安静时而活跃，时而理性时而感性。我拒绝被标签定义，我就是我。"

**超能力**：融合力 🔮 适应力 🌊 独特视角 🌟
**成长盲区**：自我探索之路漫长，有时候迷茫
**学习style**：可能是全类型，根据具体组合决定

**🦄 独角兽专属课程推荐**：
- **IB DP**：跨学科课程（如果你选IB，文理兼修是独角兽的特权）
- **AP**：根据具体兴趣选择2-3门
- **A-Level**：建议1门艺术 + 1门科学 + 1门社会科学
- **BC**：探索阶段选课更灵活

**🦄 选课建议**：
你可能是最需要"探索"的人。建议选能同时满足多个兴趣的跨学科课程，给自己更多可能性。

**🦄 报考竞争力**：个人项目、艺术+学术混合作品集
**适合社团**：多个社团都适合你，探索中找到自己的定位
